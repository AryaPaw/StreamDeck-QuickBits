Playback (now playing, play/pause, next/previous) works via the local Spotify desktop app — no API keys required.

Like requires Spotify Web API authorization below. Like checks the Liked Songs library of the authorized account — it must match your Spotify desktop login.

How to get Spotify API credentials:

  1. Go to developer.spotify.com/dashboard
  2. Log in with your Spotify account
  3. Click Create app
  4. Fill in:
    • App name: QuickBits
    • App description: Stream Deck plugin
    • Redirect URI: http://127.0.0.1:5789/callback
  5. Check Web API checkbox
  6. Click Save
  7. Go to Settings to find your Client ID and Client Secret
