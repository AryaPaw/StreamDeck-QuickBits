Connect Spotify
Note: If using a VPN, temporarily disable it during setup.
  1. Open Spotify Developer Dashboard
  2. Create a new app
  3. Set Redirect URI to http://127.0.0.1:5789/callback
  4. Select Web API in "APIs used"
  5. Save the app and open Settings
  6. Copy Client ID and Client Secret below

Connected!
